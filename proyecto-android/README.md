# Proyecto Android

Aplicacion Android nativa en Java para la evidencia GA8-220501096-AA2-EV02.

## Como abrir

1. Abrir Android Studio.
2. Seleccionar `Open`.
3. Elegir esta carpeta:

```text
C:\Users\Usuario\Documents\ActividadAndroidStudio\proyecto-android
```

4. Esperar la sincronizacion de Gradle.
5. Ejecutar la app en un emulador o telefono.

## Como generar APK

En Android Studio:

1. Abrir `Build`.
2. Seleccionar `Build Bundle(s) / APK(s)`.
3. Seleccionar `Build APK(s)`.
4. Copiar el APK generado a la carpeta `entrega`.

## Capas

- Presentacion: `app/src/main/java/com/sena/actividadandroidstudio/ui`.
- Logica de negocio: `app/src/main/java/com/sena/actividadandroidstudio/domain`.
- Acceso a datos: `app/src/main/java/com/sena/actividadandroidstudio/data`.
- Modelo: `app/src/main/java/com/sena/actividadandroidstudio/model`.
